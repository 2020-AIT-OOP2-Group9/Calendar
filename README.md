# Calendar
予定を追加できるカレンダーを3ヶ月分作成する

カレンダー表示画面で日付をクリックして、予定の追加・削除画面に移動する際、日付を判別するための方法が未定

# 実行方法
- Calendarフォルダ内で python main.py を実行
- Flask使用、バージョン 1.1.2
